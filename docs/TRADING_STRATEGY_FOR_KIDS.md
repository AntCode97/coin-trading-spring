- 사람이 컴퓨터 앞에 안 있어도 혼자서 알아서 거래해요

**비유로 이해하기:**

> - 손님이 오면 자동으로 물건을 팔아요
> - 재고가 부족하면 자동으로 주문해요
> - 우리 로봇도 이렇게 혼자 일해요!

## 3. 로봇이 하는 일 4가지 🎯

### 방법 1: 매일 조금씩 사기 (DCA) 📅

**쉬운 설명:**
> 🍱 매일 도시락 먹기랑 같아요!
> - 매일 12시에 점심을 먹죠?
> - 이 로봇도 매일 정해진 시간에 조금씩 코인을 사요

**어떻게 작동하나요?**
```
📅 시간: 8시간마다 한 번씩
💰 금액: 10,000원씩
🎯 목표: 오랫동안 조금씩 모아서 부자 되기!

예시:
- 월요일 12시: 비트코인 10,000원 어치 삼
- 화요일 12시: 이더리움 10,000원 어치 삼
- 수요일 12시: 리플 10,000원 어치 삼
...
```

**언제 좋을까요?**
- 코인 가격이 올라갈 때 좋아요
- "올라갈지 내려갈지 모르겠다!" 할 때 좋아요

---

### 방법 2: 그물망으로 잡기 (GRID) 🕸️

**쉬운 설명:**
> 🎣 낚시 그물을 여러 개 치는 것과 같아요!
> - 1층, 2층, 3층... 여러 층에 그물을 쳐두고
> - 물고기(코인 가격)이 지나가면 잡아요

**어떻게 작동하나요?**
```
가격: 100원이라고 해요

📈 가격이 올라가면 팔아요:
- 101원 → 팔기 (小 이익)
- 102원 → 팔기 (中 이익)
- 103원 → 팔기 (大 이익)

📉 가격이 내려가면 사요:
- 99원 → 사기 (싸게 사기!)
- 98원 → 사기 (더 싸게!)
- 97원 → 사기 (완전 싸게!)
```

**언제 좋을까요?**
- 가격이 왔다 갔다 할 때 (횡보장)
- "올라도 많이 안 올라갈 것 같은데?" 할 때

---

### 방법 3: 고무줄 튕기기 (평균 회귀) 🎀

**쉬운 설명:**
> 🎀 고무줄을 잡아당겼다 놓으면 다시 돌아오죠?
> - 코인 가격도 너무 올라가거나 내려가면
> - 다시 평균으로 돌아오려고 해요
> - 이걸 이용해서 돈을 벌어요!

**어떻게 작동하나요?**
```
평균 가격: 100원

📈 너무 비싸면 (120원):
- "너무 비싸! 나중에 싸지면 사야지"
- 기다렸다가 싸지면 사요

📉 너무 싸면 (80원):
- "와! 싸다! 싸울 때 사야지!"
- 지금 사서 나중에 비싸지면 팔아요
```

**핵심 도구들:**
1. **Z-Score (얼마나 특이한지)**
   - 1.2 이상이면 "와! 특이하다!" → 사거나 팔기
   - 평균이랑 얼마나 다른지 숫자로 나타낸 거예요

2. **RSI (힘 남았는지)**
   - 25 이하 = "힘 다 빠짐" → 싸서 사기 좋음!
   - 75 이상 = "너무 힘 센 것 같은데?" → 비싸서 팔기 좋음

3. **볼린저 밴드 (통로)**
   - 코인 가격이 움직이는 통로예요
   - 통로 아래에 있으면 싼 거, 위에 있으면 비싼 거

**언제 좋을까요?**
- 가격이 왔다 갔다 할 때
- 뉴스 없이 그냥 오르내릴 때

---

### 방법 4: 펌프 감지하고 승차차 타기 (VOLUME SURGE) 🚀

**쉬운 설명:**
> 🎢 놀이공원 롤러코스터 생각해보세요!
> - 갑자기 사람이 많아지면 롤러코스터가 빨리 도는데
> - 이걸 미리 감지하고 타는 거예요!
> - 하지만 조심해야 해요! 위험할 수 있거든요!

**어떻게 감지하나요?**
```
🔍 감지기 4가지:

1. 거래량이 갑자기 150% 이상 ↑
   "어? 갑자기 사람이 많아지네?"

2. 가격이 올라가는 중 (RSI 80 이하)
   "아직 오를 수 있겠는데!"

3. 점수 계산 (40점 이상)
   - 거래량: 최대 35점
   - 가격 상승: 최대 25점
   - 사고 싶은 사람: 최대 15점
   - 힘 상태: 최대 15점
   - 추세: 최대 10점

4. AI가 뉴스 확인
   "이게 진짜 좋은 정보인가 아니면 속이는 건가?"
```

**리스크 관리 (안전장치):**
```
🛡️ 손절 (-3%): 너무 손해 보면 팔아요
🎯 익절 (+6%): 조금 이득 보면 팔아요
🎢 트레일링: 3% 올라가면, 그 뒤로는 1.5% 내려가면 팔아요
⏰ 타임아웃: 60분 지나면 무조건 팔아요
```

**언제 좋을까요?**
- 뉴스가 나와서 갑자기 사람들이 많아질 때
- "오늘 뜨는 코인 뭐야?" 할 때

**주의할 점:**
- 세력이 속일 수 있어요 (펌프앤덤프)
- AI가 뉴스를 읽고 확인해요

---

### 방법 5: 세력 코인 승차차 (MEME SCALPER) 🎰

**쉬운 설명:**
> 🎰 카지노에서 세력들이 돈을 걸면
> - 그 옆에 붙어서 같이 타는 거예요!
> - 하지만 굉장히 위험해요! 빨리 빠져야 해요!

**어떻게 작동하나요?**
```
🎯 펌프 감지:
- 거래량이 5배 ↑ (500%)
- 가격이 3% ↑
- 사려는 사람이 매수 쪽으로 치우쳐짐

🏃‍♂️ 아주 빨리 움직여요:
- 5초마다 확인
- 손절: -1.5% (매우 타이트!)
- 익절: +3%
- 타임아웃: 15분
```

**대상 코인:**
```
❌ 제외 (너무 커서):
- BTC, ETH, XRP, SOL (대형 코인들)

✅ 타겟 (중소형 코인):
- 거래대금 10억~500억
- 상장된 코인만
```
